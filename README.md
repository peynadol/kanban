# Kanban Board (Work in Progress)

An interactive Kanban board built with React.  
Supports multiple boards, drag-and-drop functionality, and dynamic task management.

Currently a work in progress — actively extending features and improving UI.

---

## Features (Current)

- Multiple board support
- Columns: To Do / Doing / Done
- Drag-and-drop tasks using DnD Kit
- Add new tasks (with form validation using React Hook Form + Zod)
- Responsive layout (basic)
- State management using React hooks

---

## Planned Features / Improvements

- Task editing
- Reordering tasks within columns
- Delete tasks
- More polished styling and animations
- Full mobile responsiveness
- Persisting data (localStorage or backend)
- Board deletion confirmation dialogs

---

## Tech Stack

- React
- DnD Kit
- React Hook Form + Zod
- Shadcn
- Tailwind CSS
- Vite
- TypeScript

---

## Setup Instructions

```bash
# Clone the repo
git clone https://github.com/YOUR_GITHUB_USERNAME/kanban-board.git

# Install dependencies
npm install

# Run the app locally
npm run dev
```
